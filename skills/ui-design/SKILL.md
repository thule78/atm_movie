# UI Design Skill

## Goal
Implement the client-provided light theme accurately.

## Theme rules
- Background: white
- Primary actions: red
- Inputs/background accents: soft gray
- Text: dark gray to near black

## UI behavior rules
- Respect spacing and rounded button style from client screens.
- Prefer reusable widgets for repeated UI patterns.
- Do not redesign away from the client layout without explicit approval.
